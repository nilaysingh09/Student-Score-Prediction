# Student Score Prediction

This is a beginner machine learning project using Linear Regression.

## Objective
Predict student scores based on study hours.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Description
This project trains a Linear Regression model to understand the relationship between study hours and student scores, and predicts scores for new inputs.

## Output
Example: If a student studies for 9 hours, the model predicts their score.
